Как это работает

Три шага. Один раз мастером —
дальше автоматически.

1
📝

Мастер задаёт правила

В простом редакторе записывает 5–10 правил которые держит в голове: «если варикоз — убрать прессо», «если IMT < 22 — меньше LPG», «курсовицам — скидка 50%».

«Один раз — потом работает с каждой»
2
⚙️

AI собирает курс

За секунду по каждой клиентке: смотрит InBody, цель, противопоказания, статус оплаты — применяет правила. Мастер видит готовый черновик, одобряет или правит.

«Цена пересчиталась с 44 800 до 17 800»
3
📱

Клиентка получает план

В Telegram открывается приложение с её программой, расписанием, прогрессом. Через месяц после курса — напоминание о новом замере. Кнопка «Записаться» — звонок мастеру.

«Не нужно скачивать. Открыла — увидела»
Что внутри

Не одно «приложение для записи». Семь вещей под клиентку.

Каждая клиентка получает свою версию курса — состав, расписание, цена, акции, напоминания. Не «вилка из трёх пакетов», а реальная персонализация.

Состав курса

Какие процедуры, сколько раз, какие убрать по противопоказаниям.

Расписание

Сочетаемость процедур. Не ставит массаж лица в день LPG.

Цена

Курсовая скидка 50% только для оплативших — мастер видит сразу.

Калории и питание

Расчёт по сухой массе из InBody — не по росту и весу.

Retention-сообщения

30, 60, 90 дней после курса — сама пишет, без обзвонов админами.

Telegram-акции

Только что подходит её плану. Не «всё подряд», а под цель.

Промежуточные цели

Мини-вехи на 2-й, 4-й, 6-й неделе — клиентка видит прогресс.

Апселл моменты

Не сейчас, а когда осмысленно — на 4-й неделе LPG-абонемент.
