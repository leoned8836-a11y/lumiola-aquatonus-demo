# Lumiola Forecast

**AI-приложение для центра коррекции фигуры Aquatonus (Калуга)** — клиентский путь от первого InBody-замера до повторных продаж.

## Live Demo

👉 **[Открыть демо](https://leoned8836-a11y.github.io/lumiola-aquatonus-demo/)**

## Структура (8 экранов, 3 фазы)

### Phase 1 — Pre-purchase
Клиентка делает бесплатный InBody → за 5 минут видит персональный план + симуляцию результата → покупает курс.

### Phase 2 — During course (апселл)
Еженедельные рекомендации на основе её прогресса. Smart-апселлы LPG, прессотерапии, косметолога.

### Phase 3 — Post-course (LTV)
Maintenance plan, ранняя бронь следующего сезона, премиум-апгрейд.

## ROI калькулятор

Интерактивный экран где владелец центра вводит свои цифры и видит прогноз прироста в 3 сценариях (консервативный / реалистичный / оптимистичный).

## Stack

HTML + Tailwind CSS (CDN) + Inter (Google Fonts). Mobile-responsive. PDF-экспорт через `window.print()`.

## Использование

- Открыть в браузере (мобильный или десктоп)
- Нажать «📄 PDF» для экспорта в PDF
- Прокрутить все 8 экранов
- Перетащить slider на Screen 4 (forecast)
- Изменить цифры на Screen 8 (ROI калькулятор)
